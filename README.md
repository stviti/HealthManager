## This is a sample ASP.NET Core project using **Clean Architecture** and **CQRS** design pattern.


### Technologies used:

- Entity Framework Core
- Identity
- AutoMapper
- MediatR
- FluentValidation
- xUnit, Moq & Shouldly


### Features: 
- Clean Architecture
- CQRS design pattern
- Repository design pattern
- Custom Exceptions
- Global Error Handling
- Global Query Filters
- Custom Extension Methods
